#Focus Timer

-- Contador que contém: 

- Cards que emitem um som e cor diferentes ao serem selecionados


- Botões com as seguintes funcionalidades:

      - Play   : aciona o timer;
      - Stop   : para o timer;
      - +    : aumenta em mais 5 minutos o tempo do timer;
      - -    : diminui em 5 minutos o tempo do timer.
